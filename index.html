Apigee Entity Sample App

This app show the basics of working with entities using the Apigee JavaScript SDK. Click each button to execute an API call and see its response. For this app to work properly you must run the create step first.

To view the code and more detailed comments on how this app operates, open js/index.js.

  1. Create a 'book' entity
    This creates a new entity of type 'book'. The entity is automatically added to the 'books' collection. If you don't have a 'books' collection, it will automatically be created for you when the entity is created.
